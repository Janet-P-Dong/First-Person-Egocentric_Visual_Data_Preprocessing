# First-Person-Egocentric_Visual_Data_Preprocessing

Research collection and method protocol for first-person/egocentric visual data preprocessing, cloud classification, and future AI/AR glasses workflows.

The goal is to organize current datasets, benchmark papers, model families, cloud architecture patterns, and review procedures for first-person video captured by AI glasses, AR glasses, wearable cameras, and mobile devices.

The longer-term research framing is that egocentric video may become a foundational data layer for understanding real-world physiological, emotional, biological, and developmental change in context.

## Main Collection

- [Egocentric First-Person Video: Literature Review, Trends, and Pipeline Protocol](docs/egocentric_first_person_video_pipeline_review.md)
- [Hardware-to-Ready Dataset Flowchart](docs/hardware_to_ready_dataset_flowchart.md)

This collection covers:

- Core datasets and benchmarks such as EPIC-KITCHENS, Ego4D, Ego-Exo4D, and MECCANO.
- Feature extraction, hand-object interaction, temporal segmentation, and multimodal classification.
- Edge and cloud pipeline design for fast upload-time processing.
- Privacy, consent, and governance considerations for wearable first-person video.
- Legal-first review points for capture, upload, model inference, dataset release, and deletion.
- A standardized literature-review process for keeping the collection up to date.
- Extensions for linking visual-context changes with emotion, physiology, and biological state changes.

## Scope

This repository is intentionally focused on first-person/egocentric visual data. Local dissertation notes, APA formatting guides, raw bibliographic exports, generated outputs, PDFs, Word files, and private research materials are excluded from the GitHub repo.

## Suggested Use

Use the protocol as a starting point when designing:

- A literature review of egocentric vision and AI/AR glasses.
- A preprocessing pipeline for first-person video upload and cloud classification.
- A benchmark map for methods such as temporal action segmentation, VLM classification, and hand-object interaction.
- A research design linking visual context to physiological, emotional, and developmental time series.
