## Document Format Checklist

- Use 1 inch margins.
- Use a readable APA-accepted font. Common choices include 12-point Times New Roman, 11-point Calibri, 11-point Arial, 11-point Georgia, 10-point Lucida Sans Unicode, or 10-point Computer Modern.
- Double-space the whole paper, including references and annotations.
- Remove extra spacing after paragraphs.
- Put page numbers in the top-right header on every page.
- Use 0.5 inch first-line indentation for ordinary body paragraphs.
- Use a separate References page after the main text.
- Center and bold the page title `References`.
- Reference entries use hanging indentation, usually 0.5 inch.

## Student Title Page Pattern

Use a student title page unless the instructor or target outlet asks otherwise.

Typical order:

1. Paper title, bold, centered, title case.
2. Author name.
3. Institutional affiliation.
4. Course number and course name, if relevant.
5. Instructor name, if relevant.
6. Due date, if relevant.

For manuscript or dissertation materials, adapt this to the required university format.

## Headings

APA 7 has five heading levels. Use only as many levels as the document needs.

| Level | Format |
|---|---|
| 1 | Centered, bold, title case |
| 2 | Flush left, bold, title case |
| 3 | Flush left, bold italic, title case |
| 4 | Indented, bold, title case, ending with a period. Text continues on the same line. |
| 5 | Indented, bold italic, title case, ending with a period. Text continues on the same line. |

For our research foundation documents, Level 1 and Level 2 will usually be enough.

## In-Text Citations

Use author-date citation.

Parenthetical:

```text
(Feldman, 2012)
```

Narrative:

```text
Feldman (2012) argued that ...
```

For direct quotations, include a specific locator:

```text
(Feldman, 2012, p. 42)
```

For paraphrases, page numbers are optional but useful when the source is long or complex.

For three or more authors, use `et al.` in in-text citations:

```text
(Reindl et al., 2018)
```

For secondary citations, avoid them when possible. Find and cite the original source. If the original cannot be obtained, use `as cited in` and include only the source you actually read in the reference list.

```text
(Original Author, Year, as cited in Source You Read, Year)
```

## Reference List Core Rules

Every reference should answer four questions:

1. Who created it?
2. When was it published?
3. What is it called?
4. Where can the reader retrieve it?

General reference structure:

```text
Author, A. A. (Year). Title of work. Source. DOI or URL
```

Use a DOI when available. Prefer DOI links in URL form:

```text
https://doi.org/xxxxx
```

Do not include database names for ordinary academic database articles unless the database is the only recoverable source or the content is uniquely database-specific.

Use `n.d.` when no date is available.

Use retrieval dates only for pages designed to change over time.

## Journal Article Template

```text
Author, A. A., Author, B. B., & Author, C. C. (Year). Title of article. Journal Title, volume(issue), page range or article number. https://doi.org/xxxxx
```

Notes:

- Article title is sentence case.
- Journal title and volume are italicized.
- Issue number is in parentheses and not italicized.
- Include article number when the article uses one instead of page range.
- If a PDF has page numbers, use those page numbers for direct in-text quotations even when the reference uses an article number.

## Book Template

```text
Author, A. A. (Year). Title of book: Subtitle in sentence case. Publisher. https://doi.org/xxxxx
```

Notes:

- APA 7 does not include place of publication.
- Do not distinguish print books from ebooks unless the format matters, such as audiobooks.
- Include DOI or URL when available.

## Chapter in Edited Book Template

```text
Author, A. A., & Author, B. B. (Year). Title of chapter. In E. E. Editor & F. F. Editor (Eds.), Title of book (pp. xx-xx). Publisher. https://doi.org/xxxxx
```

## Website Template

```text
Author, A. A. (Year, Month Day). Title of page. Site Name. URL
```

Notes:

- Use the full page URL, not just the site home page.
- Use the date for the specific page, not the site-wide copyright date.
- Use `n.d.` if no date is available.
- Include a retrieval date only if the page is expected to change.

## PDF Template

Treat PDF as a file format, not a source type. Identify what the PDF actually is: article, report, book chapter, dissertation, government document, etc.

For an online report-like PDF:

```text
Author, A. A. (Year, Month Day). Title of document. Site Name. URL
```

Use a direct `.pdf` URL when it is stable and useful; otherwise use the landing page where the PDF can be downloaded.

## Annotated Bibliography Rules

The annotated bibliography follows ordinary APA paper format:

- Title page if required.
- Page numbers.
- Double spacing.
- Alphabetized reference entries.
- Hanging indentation for each reference entry.
- Annotation begins as a new paragraph below the reference entry.
- Annotation is indented 0.5 inch from the left margin so it aligns with the hanging indent.
- No extra spacing between entries unless required by an instructor.

The local sample document uses this structure:

```text
APA reference entry
    Annotation paragraph summarizing, evaluating, or explaining how the source supports the paper.
```

For this project, each annotation should usually include:

1. One-sentence summary of the paper.
2. How it supports the developmental calibration argument.
3. The limitation or caution.
4. Which construct it supports: caregiving, RJA, IJA, dyadic synchrony, neural directionality, heterogeneity, longitudinal change, or method.

## APA Style for Our Research Foundation

When converting the core-paper foundation into APA-style writing:

- Use citations for every claim that summarizes a literature cluster.
- Prefer synthesis citations over citation piles.
- Use one citation sentence for consensus claims, then explain the theoretical move in your own words.
- Distinguish broad consensus from unresolved gap.
- Avoid unsupported absolutes such as "no studies have examined."
- Prefer calibrated language:
  - "Prior work suggests..."
  - "Existing studies provide component evidence..."
  - "Less is known about..."
  - "The present study tests whether..."

## Plagiarism, Paraphrasing, and Quotations

Use citations whenever using another source's ideas, claims, theory, methods, data, exact wording, figures, tables, or distinctive framing.

Avoid patchwriting: do not keep the original sentence structure and only swap words. A real paraphrase changes the structure and wording while preserving the meaning and includes a citation.

For paraphrases:

- Cite the source even when no words are copied.
- Add page or paragraph numbers when the source is long, complex, or when a reader would benefit from the locator.
- Use synthesis when multiple papers support the same point rather than summarizing one paper at a time.

For direct quotations:

- Use quotation marks for short quotations.
- Include author, year, and page number or other locator.
- Use block quotation formatting for long quotations only when the exact wording matters.
- Prefer paraphrase in literature reviews unless the original wording is theoretically important or being analyzed.

For our dissertation literature review, default to paraphrase and synthesis. Use direct quotation sparingly for definitions, theory statements, coding manuals, or methodological language that must remain exact.

## Spelling, Hyphenation, and Abbreviations

Use consistent spelling throughout the document. Check discipline-specific terms against the source literature and APA style when uncertain.

Hyphenate compound modifiers when they come before a noun and ambiguity is possible:

```text
parent-child interaction
child-led coordination
task-specific scaffolding
```

Do not over-hyphenate familiar terms when APA or field usage treats them as open compounds.

For abbreviations:

- Spell out the term at first use, followed by the abbreviation in parentheses.
- Use the abbreviation only if it appears at least three times or is more familiar than the full form.
- Do not introduce unnecessary abbreviations for readability.
- Use standard abbreviations consistently: RJA, IJA, fNIRS, EEG, MVGC, and APA.

Example:

```text
Responding to joint attention (RJA) indexes the child's ability to follow another person's attentional bid.
```

## Bias-Free and Inclusive Language

Use person-first or identity-first language according to the preference of the community being discussed, the source literature, or participant/community guidance.

Avoid language that treats groups as deficits, labels, or homogeneous categories. Be precise about age, disability, neurodevelopmental status, socioeconomic context, race, ethnicity, language background, and caregiver role.

Prefer:

```text
children with autism
autistic children
families with low income
children from low-income households
caregivers
parents and caregivers
```

Choose the form that matches the study context and community preference; do not switch forms casually within the same document.

Avoid implying that the mother is always the caregiver unless the study specifically sampled mothers. Use `caregiver`, `parent`, `mother`, `father`, or `parent-child dyad` according to the actual sample.

## Grammar and Style

### Active and passive voice

Use active voice when the actor matters:

```text
Caregivers scaffolded the task.
```

Use passive voice when the actor is unknown, irrelevant, or when the object of study should be emphasized:

```text
Joint attention was coded from the free-play task.
```

### Anthropomorphism

Avoid giving human actions to studies, tables, models, or results.

Prefer:

```text
The results suggest...
The model estimates...
The study found...
```

Use caution with phrases like:

```text
The literature argues...
The table shows...
The data say...
```

These can be acceptable in informal drafting, but final APA-style prose should name the human authors or the analytic result when precision matters.

### Logical comparisons

Make comparisons parallel and explicit.

Avoid:

```text
Children in the structured task showed more RJA than free play.
```

Prefer:

```text
Children in the structured task showed more RJA than children in the free-play task.
```

### Verb tense

Use past tense for specific completed methods and results:

```text
We coded RJA from the structured task.
The model showed a stronger caregiving-performance association at higher RJA.
```

Use present perfect for areas of continuing inquiry:

```text
Researchers have examined parent-child synchrony using fNIRS hyperscanning.
```

Use present tense for stable conclusions, definitions, and the argument of the current paper:

```text
RJA refers to the child's response to another person's attentional bid.
This study tests whether child joint attention calibrates caregiving effectiveness.
```

### First-person pronouns

Use first person when describing the authors' actions to avoid awkward passive constructions:

```text
We tested whether RJA moderated the association between caregiving and task performance.
```

For dissertations, follow the committee or university preference if they prefer less first person.

### Singular they

Use singular `they` for a person whose gender is unknown, irrelevant, or nonbinary, and to avoid awkward constructions such as `he or she`.

### Plural and possessive nouns

Check plural and possessive forms carefully in developmental writing:

```text
children's performance
caregivers' behaviors
parent-child dyads' synchrony
the child's RJA
the parents' reports
```

Do not use an apostrophe for simple plurals:

```text
DOIs
PDFs
R scripts
```

## Tables

Use tables when exact values or structured comparisons matter. Do not duplicate the same information in both text and table.

APA-style table checklist:

- Table number appears above the table in bold: `Table 1`.
- Table title appears below the number in italic title case.
- Use clear column headings.
- Keep rules minimal; avoid vertical lines.
- Put notes below the table when needed.
- Define abbreviations in table notes unless they are standard and already defined nearby.
- Refer to every table in the text.

For this project, good table candidates include:

- Core construct map.
- Hypothesis-to-measure matrix.
- Paper cluster to dissertation argument matrix.
- Missing PDF/DOI tracking table.
- APA reference audit table.

## Figures

Use figures when relationships, processes, models, or patterns are easier to understand visually.

APA-style figure checklist:

- Figure number appears above the figure in bold: `Figure 1`.
- Figure title appears below the number in italic title case.
- Place a note below the figure if needed to explain abbreviations, symbols, or source/adaptation information.
- Refer to every figure in the text.
- Make labels readable and self-contained.
- If adapted from another source, include appropriate attribution and permission status when required.

For this project, good figure candidates include:

- Developmental calibration model.
- Caregiving to outcome pathway moderated by RJA/IJA.
- Parent-to-child and child-to-parent neural directionality diagram.
- Structured task versus free-play comparison model.

## Final APA Review Pass for Project Writing

Before sending any APA-style document out:

1. Check every literature claim has a citation.
2. Check every in-text citation appears in the reference list and every reference is cited in text.
3. Replace citation piles with synthesis where possible.
4. Verify all DOIs and URLs.
5. Check heading levels.
6. Check tense consistency.
7. Check bias-free and caregiver-role language.
8. Check abbreviations were introduced and used consistently.
9. Check tables and figures are numbered, titled, cited in text, and explained.
10. Check quotation marks and page numbers for any direct quotes.
